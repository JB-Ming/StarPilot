# 範例銷售數據

這是一個測試用的 Excel 檔案範例。

## 使用方式

1. 準備你自己的 Excel 檔案，放到這個資料夾
2. 或者對 AI 說：「幫我生成一個範例銷售數據檔案」

## 範例對話

```
👤：幫我建立一個範例 Excel，包含：
    - 日期（2024 年 1-12 月）
    - 產品名稱（手機、筆電、平板、手錶）
    - 銷售數量
    - 單價
    - 總金額

🤖：好的，我會建立一個包含 100 筆資料的範例檔案...
    [生成完成] 檔案已儲存為 sample_sales.xlsx
```

---

放入你自己的資料檔案後，就可以開始對 AI 說：

- 「分析這個 Excel 的銷售趨勢」
- 「找出賣最好的前 5 個產品」
- 「畫出每月銷售額的折線圖」
