# 📦 套件說明文件

## 已預先安裝的套件

當你啟動 Codespaces 或執行安裝腳本後，以下所有套件都已經可以使用：

### 📊 資料處理與分析
- **pandas** - 處理 Excel、CSV 等表格資料
- **numpy** - 數值計算
- **openpyxl** - 讀寫 Excel 檔案（.xlsx）

### 📈 資料視覺化
- **matplotlib** - 繪製各種圖表
- **seaborn** - 更美觀的統計圖表
- **plotly** - 互動式圖表

### 🌐 網頁應用
- **streamlit** - 快速建立互動式網頁應用
- **flask** - 建立 API 伺服器

### 🕷️ 網頁爬蟲
- **requests** - 下載網頁內容
- **beautifulsoup4** - 解析 HTML
- **selenium** - 自動化瀏覽器操作

### 📄 檔案處理
- **Pillow** - 圖片處理（縮放、裁切、轉換格式）
- **python-docx** - 處理 Word 文件
- **PyPDF2** - 處理 PDF 檔案

### ⚙️ 自動化工具
- **schedule** - 排程任務（定時執行）
- **python-dotenv** - 管理設定檔

### 🛠️ 實用工具
- **tqdm** - 顯示進度條
- **rich** - 美化終端機輸出

## 🔄 重新安裝

如果環境出問題（極少發生），可以：

1. 回到 GitHub 專案頁面
2. 點擊 **Code** → **Codespaces**
3. 找到你的 Codespace，點選「...」選單
4. 選擇「Delete」刪除
5. 重新建立一個新的 Codespace（會自動重新設置所有套件）

> 💡 所有套件會自動重新安裝，你不需要做任何手動操作
