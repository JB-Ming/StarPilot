# 🎨 Playground - 你的實驗區

這裡是你的自由創作空間！

## 💡 這裡可以做什麼？

所有你想嘗試的自動化任務都可以在這裡實驗：

### 📋 常見使用情境

1. **工作自動化**
   - 自動生成週報、月報
   - 批次處理郵件附件
   - 整理會議記錄

2. **個人專案**
   - 照片整理與備份
   - 下載資料夾清理
   - 個人理財記錄分析

3. **學習實驗**
   - 測試新的想法
   - 練習與 AI 對話
   - 嘗試複雜的組合任務

---

## 🚀 如何開始？

### 方法 1：直接對話
```
按 Ctrl+I 開啟 Copilot Chat

👤：我想在 playground 建立一個自動化工具，可以...

🤖：好的，我會在 playground/ 資料夾建立...
```

### 方法 2：從範例修改
1. 複製 `examples/` 中的任何範例到這裡
2. 告訴 AI 你想改成什麼樣子
3. 讓 AI 幫你修改

---

## 📝 建議的資料夾結構

你可以這樣組織：

```
playground/
├── work/              # 工作相關的自動化
├── personal/          # 個人專案
├── experiments/       # 實驗性質的測試
└── archive/          # 不再使用但想保留的
```

或者完全按照你的方式組織！
